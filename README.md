# [Re]Hookify site

[Re]Hookify uses [Next.js](https://nextjs.org/) for site and documentation.

## Getting Started

- fork and clone this repo to your computer
- install `pnpm`

```bash
npm i -g pnpm
```

- install dependencies

```bash
pnpm i
```

- run development server

```bash
pnpm run dev
```
