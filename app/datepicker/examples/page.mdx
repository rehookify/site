# Examples

Tech stack for all examples: Vite, React, TailwindCSS, TypeScript

1. [Single Date](https://codesandbox.io/p/sandbox/rehookify-all-in-one-month-calendar-176kep) - An all-in-one solution utilizing dates, months, and years. The calendar's starting day is set to Monday.
2. [Multiple Date](https://codesandbox.io/p/sandbox/multiple-calendars-and-dates-dp-context-140xny) - An all-in-one solution using `DatePickerProvider` alongside `useDatePickerContext`. It uses dates, months and years, also, providing multiple dates selection and dates toggling.
3. [Range Picker](https://codesandbox.io/p/sandbox/range-with-context-hooks-fyv72j) - A modular solution that utilizes `DatePickerStateProvider` and context hooks. It offers multiple calendars with predefined minimum and maximum dates, set in range mode.
4. [Time Picker](https://codesandbox.io/p/sandbox/time-picker-with-multiple-dates-lf36qc) - A modular solution featuring a multi-date time picker with locales configured to display the 12-hour format.
